# Project Structure

## data

This is where raw and preprocessed data should go.

## notebooks

This is where code for preprocessing data and training/running models is executed.

## src

This is where code for everything is written, but not necessarily run.

# To run

Code is written in `src`, then imported and used in `notebooks`.

Assuming it takes too long on your local machine, we can use Google Colab.

1. In Google Colab, click File -> Open Notebook -> Github -> THIS REPO -> THE NOTEBOOK YOU WANT TO RUN
2. Run the setup at the top
3. Run the training code in the notebook
